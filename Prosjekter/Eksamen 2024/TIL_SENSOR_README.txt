Åpning/Kjøring
--------------
1. Sørg for å ha node installert (https://nodejs.org/en)
2. Åpne prosjektmappe i VSCode via "File -> Open Folder"
3. Åpne en terminal via "Terminal -> New Terminal"
2. Kjør kommandoen "npm install" i terminalen
3. Kjør kommandoen "npm run dev -- --open" for å åpne live server


Besvarelse
----------
Koden til hver oppgave ligger i hver deres undermapper i "src/routes" i filene "+page.svelte"
.json-fil, diagrammer og andre importerte filer er i mappen "src/lib"
